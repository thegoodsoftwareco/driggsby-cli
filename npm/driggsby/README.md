# driggsby

`driggsby` is the local CLI for connecting AI clients to Driggsby over MCP.

## Quick Start

```bash
npx driggsby@latest login
npx driggsby@latest status
npx -y driggsby@latest mcp-server
```

For non-interactive MCP launcher flows, use:

```bash
npx -y driggsby@latest mcp-server
```
